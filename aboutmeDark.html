About me

Hey, all!

My name is Rowan McAlpin, and I am a genius. Or, at least, I think so. Well, anyway, I am a computer programmer. And I am 12. You heard me right. I created this site from scratch. Meaning bare code.

How this site was made

I made this site with very clear intentions in mind: an outstandingly easy-to-use layout, with a dark theme (look for the link at the bottom), that appeared to be made with a site-making webkit, like wix*.

Steps

The steps I used to make this website were quite simple:

  1. Design layout
  2. Decide content
  3. Make sitemap
  4. Code home file
  5. Create CSS files
  6. Add in all other HTML files.

Thats it!

Please note: I have not relationship whatsoever with wix. I used it as an example.

Coding career

I started coding when I was 8. My dad was showing me a website he had coded, and I knew then and there that I wanted to learn to code. Believe it or not, my best language is not HTML, though that is up there. I like Python most of all. Anyway, my dad wrote out a list of all of the basic commands, like <p>, <h1>, etc.

Once I learned those by heart, I went looking for more. Soon after, I learned my uncle also codes, so I reached out to him. He taught me some more, and let me use a sub-domain, the one I am still using today, rowan.cementhorizon.com.
